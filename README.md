### node (ノード)と tag (タグ) の違い

タグとは用途の種類

地図で言う所の
- 病院
- 警察署
- 学校

ノードは場所
- 今、何処にいる？
- 何県？何市？何町にいる？

なので
DOMは
〇〇県病院を通った後にある
〇〇市警察署を右に曲がった
〇〇町学校に
学級日誌を書き忘れた
等
